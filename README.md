# Rust & Ritueel — Statische website

Dit is een eenvoudige statische website bedoeld voor een dienst: massage aan huis kort voor het slapengaan (tot 23:00) om mensen met slaapproblemen te helpen.

Inhoud:
- index.html — hoofdpagina en content (Nederlands)
- styles.css — styling
- script.js — klein frontend script voor formulier en smooth scrolling

Achtergrondfoto's:
- Unsplash (gratis te gebruiken). De exacte links staan in de footer van de site.

Installatie en deployment:
1. Kopieer de bestanden naar een map.
2. Open `index.html` lokaal om te testen.
3. Voor live-publicatie kun je GitHub Pages, Netlify of Vercel gebruiken:
   - GitHub Pages: maak een repository, push de bestanden en schakel Pages in op de `main` branch.
   - Netlify/Vercel: sleep de map naar de dashboard of koppel repo en publiceer.
4. Als je reserveringen wilt ontvangen, koppel het formulier aan een backend of een formulierservice (Formspree, Netlify Forms, enz.).

Aanpassingen die je wellicht wilt doen:
- Vervang de placeholder-telefoon en details door jouw gegevens.
- Pas regio/werkgebied aan.
- Koppel het formulier aan je e-mail of boekingssysteem.
- Voeg echte testimonials / certificeringen toe voor extra vertrouwen.
- Eventueel privacyverklaring en algemene voorwaarden toevoegen.
